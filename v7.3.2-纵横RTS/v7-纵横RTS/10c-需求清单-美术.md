# 10c · 需求清单 · 美术（v7.3.1 · 按小时拆分）

> 工期 3.5 天 · 8 张角色立绘 + 完整 UI 风格
> 每天任务按小时列出，含验收物。**所有任务必须在指定时间内完成，否则触发砍线。**

---

## D1（全天 8h · 风格打样+草图）

### 上午（4h）

- [ ] **ART-00（0–1h）** 风格打样 mood board
  - **做什么**：1–2 张 mood board，战国冷峻色调，竹简/丝帛/青铜质感。
  - **验收物**：mood_board.png。
  - **依赖**：与策划对齐视觉世界观。

- [ ] **ART-01（1–2h）** 主界面布局线框
  - **做什么**：
    - 顶部：关键事件横幅（占 8% 高度）。
    - 主区：地图（65%）+ 三维数值条（顶部小条）。
    - 中下：事件流（15%）。
    - 底部：状态栏（12%）。
    - 召见界面 modal 弹窗布局。
  - **验收物**：layout_wireframe.png + 锚点说明 markdown。
  - **依赖**：与程序对齐 UI 占位尺寸。

- [ ] **ART-02（2–3h）** 君主立绘草图 3 张
  - **做什么**：
    - 秦王嬴稷（深色威严、雄猜）
    - 赵王赵何（浅色犹疑、踌躇）
    - 齐王田地（精明算计、渔利）
  - **验收物**：3 张主立绘草图（草图阶段，可简化）。

- [ ] **ART-03（3–4h）** 召见界面背景草图
  - **做什么**：3 套召见背景草图（秦/赵/齐王宫大殿简笔，单色 + 关键装饰元素）。
  - **验收物**：3 张背景草图。

### 下午（4h）

- [ ] **ART-04（4–5h）** 行动牌卡面 5 种草图
  - **做什么**：
    - 游说（舌/言辞符号）
    - 离间（裂痕/双刃符号）
    - 刺探（眼/窥视符号）
    - 许诺（玉/信物符号）
    - 传信（竹简/飞鸟符号）
  - **验收物**：5 张牌面草图。

- [ ] **ART-05（5–6h）** 卡牌方向选择 UI 草图
  - **做什么**：3 个方向按钮（推合纵/推亲秦/中立建言）的视觉风格（色调+图标）。
  - **验收物**：方向 UI 草图。

- [ ] **ART-06（6–7h）** 情报牌卡面草图
  - **做什么**：与行动牌区分（色调/形状不同）—— 体现"信息"感。
  - **验收物**：情报牌草图。

- [ ] **ART-07（7–8h）** 三立场问卷弹窗样式草图
  - **做什么**：3 个大按钮（A/B/C = 合纵派/中立自保派/亲秦派）+ 问题文字 + 跳过提示。
  - **验收物**：弹窗草图。

**D1 验收线**：
- mood board + 布局线框 + 3 君主立绘草图 + 3 召见背景 + 5 行动牌 + 方向 UI + 情报牌 + 三立场弹窗 全部就位。

## D2（全天 8h · 资产初版+立绘线稿）

### 上午（4h）

- [ ] **ART-08（0–1h）** 地图 view 资产
  - **做什么**：
    - 三节点底图（秦/赵/齐 + 名字 + 三维数值条占位）。
    - 路径线（秦↔赵、赵↔齐，不直连秦↔齐）。
    - 玩家位置图标（纵横家形象剪影）。
  - **验收物**：map_view.png + 节点 PNG。

- [ ] **ART-09（1–2h）** 国家状态标签 5 种状态
  - **做什么**：
    - `谈判中·第1轮`（蓝色 + 脉冲）
    - `谈判中·第2轮`（黄色 + 脉冲）
    - `召见`（绿色 + 强烈脉冲高亮）
    - `决策已定`（灰色）
    - `已处理`（深灰色）
  - **验收物**：state_labels.png（5 个状态对比）。
  - **依赖**：与程序对齐状态机名称。

- [ ] **ART-10（2–4h）** 君主立绘线稿 3 张
  - **做什么**：
    - 秦王线稿（深色服饰、严肃表情）
    - 赵王线稿（浅色服饰、眉头微皱）
    - 齐王线稿（暖色服饰、玩味表情）
  - **验收物**：3 张线稿（半身胸像）。

### 下午（4h）

- [ ] **ART-11（4–5h）** 5 近臣立绘/剪影
  - **做什么**：
    - 张仪（秦·连横推手，深色精明）
    - 魏冉（秦·务实扩张，粗犷武将）
    - 平原君（赵·主合纵，儒雅文臣）
    - 廉颇（赵·主战不信秦，老将威武）
    - 孟尝君（齐·谨慎渔利派，老谋深算）
  - **验收物**：5 张近臣立绘或剪影（半身像）。
  - **依赖**：[11a · 美术设计稿](11a-角色设计稿-美术.md) 的视觉关键词。

- [ ] **ART-12（5–6h）** 召见背景线稿 3 张
  - **做什么**：
    - 秦·咸阳殿（黑红主调，兵甲陈列）
    - 赵·邯郸殿（青灰主调，简朴紧张）
    - 齐·临淄殿（暖黄主调，奢华富贵）
  - **验收物**：3 张召见背景线稿。

- [ ] **ART-13（6–7h）** 行动牌卡面初版 5 种
  - **做什么**：
    - 统一牌框风格（竹简/木牍形态）。
    - 5 种区分：游说（舌）、离间（裂痕）、刺探（眼）、许诺（玉）、传信（竹简）。
    - 每张牌：牌名 + 简短效果提示。
  - **验收物**：5 张行动牌 PNG。
  - **依赖**：D1 草图。

- [ ] **ART-14（7–8h）** 情报牌卡面初版
  - **做什么**：与行动牌区分（色调冷一点，纸面纹理）。
  - **验收物**：情报牌 PNG。

**D2 验收线**：
- 3 君主线稿就位。
- 5 近臣立绘就位。
- 3 召见背景线稿就位。
- 5 行动牌 + 情报牌 + 方向 UI + 三立场弹窗初版就位。
- 地图 view + 状态标签就位。

  - **验收物**：3 张背景终版 PNG。

- [ ] **ART-17（3–4h）** 5 近臣立绘终版
  - **做什么**：上色 + 精修。
  - **验收物**：5 张近臣立绘终版。

### 下午（4h）

- [ ] **ART-18（4–5h）** 数值条 UI 终版
  - **做什么**：
    - 玩家三维：合纵青绿 / 名望金黄 / 心计紫。
    - 各国三维在节点上方显示（小型横向排布）。
    - 警戒区（≤ 10 / ≥ 90）红色标记。
    - 值变动时的短暂闪烁/跳动动效（提供动画参数给程序）。
  - **验收物**：数值条 PNG + 动画参数表。
  - **依赖**：与程序对齐动画时长。

- [ ] **ART-19（5–6h）** 事件流样式终版
  - **做什么**：
    - 底部滚动区域，半透背景。
    - 每条事件：来源标签 + 文本。
    - 滚动速度适中，清晰可读。
  - **验收物**：事件流样式 PNG + 动画参数。

- [ ] **ART-20（6–7h）** 三立场问卷弹窗终版
  - **做什么**：
    - 3 个大按钮（A=合纵派 / B=中立自保派 / C=亲秦派）+ 问题文字。
    - 跳过提示（"5 秒不点自动选 B"）。
    - 战国主题皮肤（竹简/丝帛）。
  - **验收物**：三立场弹窗 PNG + 按钮 hover/click 状态。
  - **依赖**：与程序对齐 5 秒超时机制。

- [ ] **ART-21（7–8h）** 标题+落幕 UI 终版
  - **做什么**：
    - 标题画面：游戏标题 + 副标题 + 战国纹样 + "开始"按钮。
    - 落幕画面：全屏半透遮罩 + 评语文本居中（3 立场 × 3 终局 = 9 种模板）+ "再来一局"按钮。
  - **验收物**：标题 + 落幕 PNG。

**D3 验收线**：
- 全部立绘 + 召见背景终版就位。
- 全部 UI（数值条 / 事件流 / 三立场弹窗 / 标题 / 落幕）终版就位。
