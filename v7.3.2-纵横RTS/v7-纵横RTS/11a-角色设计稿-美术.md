# 11a · 角色设计稿 · 美术（v7.3 · 性格→立绘）

> **目的**：将每个角色的**性格**翻译为**立绘视觉关键词**，让美术能直接基于本文档产出 3 君主 + 5 近臣的主立绘、召见背景、风格统一的角色形象。
> **面向对象**：美术（D1 出草图 / D2 出线稿 / D3 出终版）。
> **不是**：程序训练 agent 用的 prompt 设计稿（见 [11b · 程序设计稿](11b-角色设计稿-程序.md)）。

---

## 总风格基调

| 项目 | 设定 |
|---|---|
| **时代感** | 战国中后期（公元前 300 年前后），服饰参考《刺客聂隐娘》《大秦帝国》等影视的战国风。 |
| **色调** | 战国冷峻为主：秦用黑红、赵用青灰、齐用暖黄。三国色系严格区分。 |
| **质感** | 竹简/丝帛/青铜为主，避免亮金亮银的廉价感。 |
| **服饰** | 君冕+深衣为主，纹样简洁不堆砌。 |
| **表情** | **静态主立绘仅 1 张**（不画多表情），通过姿态和服饰传达性格。召见时仅用**背景+头身剪影**切换。 |
| **半身像构图** | 头部到胸口为主，背景虚化，单色铺底+关键装饰。 |
| **分辨率** | 800×1200 px（半身像），召见背景 1920×1080 px。 |

## 一、3 君主主立绘

### 1.1 秦王 · 嬴稷

#### 性格档案（来自 11b + 04）

- **核心利益**：东出函谷吞六国。
- **性格关键词**：**雄猜、果决、多疑、霸气、压迫感**。
- **典型动作**：居高临下俯视、单手托腮思考、握剑柄的手。
- **代表色**：**黑红**（秦尚黑）。

#### 立绘视觉关键词

| 维度 | 关键词 |
|---|---|
| **主色调** | 黑、红、金（克制不堆砌） |
| **服饰** | 黑色深衣+黑红王冕（冕版九旒），玄甲护肩暗示军功 |
| **眼神** | 鹰视狼顾，眼角微挑，瞳孔深黑带红光 |
| **表情** | 冷峻、半笑、嘴角上扬带威胁 |
| **姿态** | 端坐或半侧身，单手扶剑柄或托下巴 |
| **背景** | 黑红主调，兵甲陈列剪影，函谷关剪影（远景） |
| **纹样** | 玄鸟（秦之图腾）、虎纹 |
| **氛围** | 阴鸷、压迫、不可一世 |
| **光影** | 顶光为主（突出眼神的压迫），侧面暗影 |
| **独特元素** | 一把横置的剑（不抽出来） |

#### 召见背景色

- **主色**：黑 + 朱红 + 暗金。
- **元素**：咸阳殿柱列、远景函谷关剪影、玄鸟纹旗帜。
- **光线**：从顶部射下的强光，地面有长长的王座影子。

---

### 1.2 赵王 · 赵何

#### 性格档案

- **核心利益**：抗秦自保，不愿独战。
- **性格关键词**：**犹疑、踌躇、柔中带刚、纠结、可被说服**。
- **典型动作**：眉头紧锁、双手交握、坐立不安。
- **代表色**：**青灰**（赵尚武但困顿）。

#### 立绘视觉关键词

| 维度 | 关键词 |
|---|---|
| **主色调** | 青、灰、白（克制朴素） |
| **服饰** | 素色深衣+简朴王冕（七旒），无过多装饰 |
| **眼神** | 略带愁意，眉头微皱，眼底有血丝暗示失眠 |
| **表情** | 忧国忧民，嘴角下垂，眉宇间有"问天下何以自处"之惑 |
| **姿态** | 端坐但身体微微前倾，**双手交握**置于膝上 |
| **背景** | 青灰主调，简朴王座，远景赵长城剪影 |
| **纹样** | 飞虎（赵之图腾）、云雷纹 |
| **氛围** | 紧绷、压抑、渴望援助 |
| **光影** | 散射光，阴影较柔，没有压迫感 |
| **独特元素** | 膝上展开的一卷竹简（地图或求援信） |

#### 召见背景色

- **主色**：青灰 + 暗白 + 微量土黄。
- **元素**：邯郸殿柱列（朴拙）、远景赵长城剪影、军旗半卷。
- **光线**：散射自然光，整体偏冷，地面有未干的雨水痕迹。

---

### 1.3 齐王 · 田地

#### 性格档案

- **核心利益**：渔利，维持均势。
- **性格关键词**：**精明、算计、玩味、油滑、坐山观虎斗**。
- **典型动作**：微笑、扇子轻摇、若有所思地拨弄玉石。
- **代表色**：**暖黄**（齐富）。

#### 立绘视觉关键词

| 维度 | 关键词 |
|---|---|
| **主色调** | 暖黄、金、橙（富贵但不庸俗） |
| **服饰** | 暖黄深衣+华贵王冕（八旒），丝绸腰带 |
| **眼神** | 笑眯眯但眼珠转动，瞳孔带精明算计的光 |
| **表情** | 似笑非笑，嘴角含玩味，眼底藏刀 |
| **姿态** | 慵懒斜倚王座，**单手执扇**或**单手把玩玉璧** |
| **背景** | 暖黄主调，奢华王座，远景临淄城剪影 |
| **纹样** | 朱雀（齐之图腾）、云纹、卷草纹 |
| **氛围** | 富贵、慵懒、暗藏锋芒 |
| **光影** | 暖光，柔光为主，半身像明亮 |
| **独特元素** | 一把折扇（半开）或一块玉璧（手心把玩） |

#### 召见背景色

- **主色**：暖黄 + 橙 + 微量金。
- **元素**：临淄殿奢华柱列、丝绸帷幔、远景繁华城市场景。
- **光线**：暖光从窗外射入，有丝绸帷幔的柔光。

## 二、5 近臣立绘/剪影

> **范围**：每近臣**1 张主立绘**（半身像，姿态简洁，表情克制）。可裁剪为剪影用于事件流、情报牌插画。
> **重要性**：近臣是 LLM 决策时的"倾向权重注入"，美术表现需让玩家能一眼识别"这位在影响哪国"。

### 2.1 张仪（秦 · 连横第一推手）

| 维度 | 关键词 |
|---|---|
| **职能** | 秦的连横核心谋士 |
| **性格** | 狡黠、口才极佳、唯结果论 |
| **视觉** | 瘦削、修长、眼神闪烁、嘴角永远带笑 |
| **服饰** | 秦黑红深衣，文士冠，**手持一卷竹简**（"连横"二字隐约可见） |
| **姿态** | 身体前倾，像在说服人 |
| **背景** | 秦黑红帷幔，远景栈道 |

### 2.2 魏冉（秦 · 穰侯 · 务实扩张）

| 维度 | 关键词 |
|---|---|
| **职能** | 秦的军事扩张推手 |
| **性格** | 粗犷、果断、重实务、军功至上 |
| **视觉** | 壮实、方脸、络腮胡（细画）、眼神锐利 |
| **服饰** | 秦黑红深衣+**半甲护肩**、武冠 |
| **姿态** | 单手叉腰或**握剑柄** |
| **背景** | 函谷关外军营、战车剪影 |

### 2.3 平原君（赵 · 主合纵）

| 维度 | 关键词 |
|---|---|
| **职能** | 赵的合纵外交推手 |
| **性格** | 儒雅、有大局观、略显书生气 |
| **视觉** | 清瘦、长脸、温和、眼神坚定 |
| **服饰** | 赵青灰深衣、文士冠、**手持玉圭** |
| **姿态** | 端正拱手，像在行礼 |
| **背景** | 赵国书斋、竹简堆 |

### 2.4 廉颇（赵 · 主战不信秦）

| 维度 | 关键词 |
|---|---|
| **职能** | 赵的武将，独战派 |
| **性格** | 刚直、有傲气、老将不服老 |
| **视觉** | 魁梧、白发、浓眉、疤痕（细画）、眼神如火 |
| **服饰** | 赵青灰深衣+**全甲护肩**、武冠、**披风** |
| **姿态** | 双手抱胸或**握长戟** |
| **背景** | 赵长城烽火台、军阵剪影 |

### 2.5 孟尝君（齐 · 谨慎渔利派）

| 维度 | 关键词 |
|---|---|
| **职能** | 齐的渔利谋主，门客众多 |
| **性格** | 老谋深算、世故、笑里藏刀 |
| **视觉** | 圆润、慈眉善目表象下眼神锐利、有留须 |
| **服饰** | 齐暖黄深衣+狐裘领、**手持算筹或玉珠** |
| **姿态** | 斜倚，单手拨算筹 |
| **背景** | 临淄市集门客剪影、账册堆 |

## 三、整体一致性与差异性矩阵

> 美术在做具体立绘时，应保持 8 位角色的视觉差异性，让玩家一眼能区分。

| 角色 | 主色 | 体型 | 关键道具 | 表情 | 整体氛围 |
|---|---|---|---|---|---|
| 秦王 | 黑红 | 壮 | 剑 | 冷峻 | 压迫 |
| 赵王 | 青灰 | 中 | 竹简 | 忧 | 紧绷 |
| 齐王 | 暖黄 | 圆润 | 扇/玉 | 玩味 | 富贵 |
| 张仪 | 黑红 | 瘦 | 竹简 | 狡黠 | 阴鸷 |
| 魏冉 | 黑红 | 壮 | 剑 | 粗犷 | 军威 |
| 平原君 | 青灰 | 瘦 | 玉圭 | 儒雅 | 文气 |
| 廉颇 | 青灰 | 壮 | 长戟 | 刚直 | 武将 |
| 孟尝君 | 暖黄 | 中 | 算筹 | 慈中带刀 | 老练 |

## 四、召见界面整体规范

> 当玩家抵达召见国时，弹出的 modal 应由 4 部分组成：

```
┌──────────────────────────────────────┐
│  [召见背景图]                          │  ← 君主所属国家的主色
│                                      │
│  [君主立绘]    [君主名 + 困局描述]      │  ← 立绘在左，文字在右
│  (半身像)                             │
│                                      │
│  [召见前情滚动条]                       │  ← 半透背景
│                                      │
│  [打字输入框]    [发送]    [取消]       │  ← 底部固定
└──────────────────────────────────────┘
```

### 4.1 颜色规则

- 召见秦王：背景黑红 + 朱红文字强调。
- 召见赵王：背景青灰 + 暗白文字强调。
- 召见齐王：背景暖黄 + 暖橙文字强调。

### 4.2 立绘位置

- 君主立绘在 modal 左侧，**高度占 modal 60%**。
- 立绘边缘淡出（fade out）到背景色，避免硬切。

## 五、立绘统一规范（D1 必出 + D2 必执行）

| 规范 | 内容 |
|---|---|
| **画幅** | 800 × 1200 px（半身像，含头部到胸口 + 一部分背景） |
| **线稿** | 6–8 px 粗细，主线明确，副线虚化 |
| **上色** | 7–9 色调色板，每个角色严格控制主色（不超过 4 色） |
| **光影** | 顶光 + 侧光组合，避免平涂 |
| **背景** | 单色铺底 + 1–2 个关键剪影元素 |
| **纹样** | 战国时期真实纹样（云雷、饕餮、玄鸟、朱雀、虎纹），不臆造 |
| **服饰参考** | 复原战国深衣形制，冕版旒数严格按等级 |
| **不画** | 多表情、多姿态、多武器；本期只画 1 张主立绘 |

## 六、近臣剪影裁剪规范（D2 必出）

为节约时间，近臣可只出**剪影版**（黑底+单色填充），用于：
- 事件流的"秦使张仪入齐"等动作的引子。
- 情报牌插画（可选）。
- 君主立绘旁的近臣伴立位。

剪影规范：
- 800 × 1200 px。
- 单色（君主所属国主色）。
- 头部轮廓 + 肩部 + 关键道具（剑/竹简/玉圭/扇等）轮廓。
- 不画表情细节。

## 九、与 11b（程序设计稿）的关系

| 11a（本文档）| 11b |
|---|---|
| 性格 → 立绘 | 性格 → agent prompt |
| 视觉关键词 | 决策关键词 |
| 给美术看的 | 给程序看的 |
| 不影响 agent 行为 | 不影响立绘 |

> **两者必须保持一致**——同一性格在两边描述要吻合，避免出现"美术画的嬴稷很温和但程序 agent 表现很凶"的撕裂。
