# 00 · 索引与说明（v7.3.1）

> **战国纵横家 × 大回合策略 × 自由行动同步博弈 × AI 群体智能 × 三立场玩家画像**
> GameJam 项目（**3.5 天** / 3 人）。每回合 = 抽行动牌 → 自由行动阶段（不限时，玩家手动推进）：你在地图上赶路、面见君主、打字游说，与此同时三位君主 Agent 在**最多 2 轮**同步博弈，局面在你眼前实时演化。每回合开始弹出三立场问题，识别玩家立场（合纵派 / 中立自保派 / 亲秦派）。

## 一句话定位

你扮演一位**战国纵横家**，每回合开局抽行动牌，随即进入**自由行动阶段**——一张大地图上，你从一个国家赶往另一个国家，观察三国君主之间正在进行的密谈、施压、结盟与背叛。当你抵达某国时，君主可能已因自主博弈而进入"**召见**"状态，等待你用**打字对话**解决它当前的困局。**每回合开始时，你会被问一个简短问题**——不是测试，是让系统理解你是谁。你不是在选择选项——你是在真实的博弈中，用自己的话撬动天下。

## 关键决策（v7.3.1 锁定）

- **3.5 天工期 / 3 人**：D1 全天设计冻结+打底；D2 核心闭环（mock）；D3 上午 LLM 接入+平衡；D3 下午-D4 集成打磨+录屏。
- **6 回合 × 2 月 = 1 年**：每回合 2 个月，回合开始结算上回合数值 → 生成本回合关键事件 → **弹出立场问卷** → 抽行动牌 → 自由行动。
- **二值判定**：成功/失败。游说/传信打牌前玩家选择方向（推合纵/推亲秦/中立信息），方向不同→数值变动方向不同。
- **面谈二值映射**：LLM 裁决结果仅"采纳"或"拒绝"二值，由 LLM 评分（0–10）>= 6 视为采纳。
- **态度识别 = 三立场问卷**：每回合 1 题（题库 6 题），3 选 1 测立场——**合纵派 / 中立自保派 / 亲秦派**。
- **自由行动不限时**：Agent 博弈 ≤ 2 轮/对。召见打字不限时。
- **情报牌后置获取**：回合初仅抽行动牌。
- **玩家三维 + 国家三维**：合纵/名望/心计 + 国威/盟信/战心。即死机制保留。
- **降级链保留**：LLM 断连 → 全局 mock 规则决策表。
