## 三、二值判定机制

```
行动成功率 = clamp(基础成功率 + 对应数值 × 加成系数, 5%, 95%)
掷骰 → 成功 / 失败
```

| 行动 | 基础成功率 | 加成来源 | 满值成功率 |
|---|---|---|---|
| 游说 | 40% | 名望 × 0.3 | 70% |
| 离间 | 35% | 心计 × 0.3 | 65% |
| 刺探 | 50% | 心计 × 0.25 | 75% |
| 许诺 | 45% | 名望 × 0.3 | 75% |
| 传信 | 40% | 合纵 × 0.3 | 70% |
| 召见面谈 | 45% | (名望+合纵)/2 × 0.3 | 75% |

**取消五档倍率**。成功 → 固定效果（按方向走）。失败 → 名望 −3（面谈失败 −5）。

### 3.1 面谈二值映射规则

```
LLM 输出：
  comprehension: 0–10
  stance_match: 0–10
  persuasion: 0–10
  verdict: 采纳 | 拒绝（建议性）

仲裁器：
  score = comprehension × 0.3 + stance_match × 0.4 + persuasion × 0.3
  score ≥ 6.0 → 采纳（按玩家游说方向走 1.5 倍）
  score < 6.0 → 拒绝（名望 −5 + 1 张拒绝原因情报牌）
```

超时 8s → 关键词匹配 mock：
- 检测"应允""好""定""可"等 → 采纳
- 检测"否""退""罢""不"等 → 拒绝
- 都不命中 → 综合分 5.0 → 拒绝

## 五、平衡校准要点

1. **三值即死平衡**：当前合纵衰减 −4、成功率上限 70%（游说 +8 一次），合纵归零需连续约 5 次失败 + 漂移，可达。名望从 50 降到 0 需 17 次失败——偏多。**建议**：名望衰减加快或失败惩罚加重。
2. **偏秦路径**：玩家选推亲秦，合纵可降到 0 即死（不可长期偏秦）。**建议**：让玩家可主动"收手"——停止推亲秦 → 合纵自然回升。
3. **心计上限**：心计自然衰减 −3，许诺成功 +6，可达上限。**建议**：心计 100 死亡的剧情是"三王震怒"，玩家应能感知风险。
