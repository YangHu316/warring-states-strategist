# 02 · 核心玩法（v7.3.1 · 6 回合 + 关键事件 + 二值判定 + 三立场问卷）

## 玩法骨架

```
┌─ 回合开始 ─────────────────────────────────────┐
│ ① 三立场问卷（1 题，必答，跳过 = 选 B=中立自保） │
│ ② 结算上回合数值 → 生成本回合"关键事件"           │
│ ③ 抽行动牌（补至 5 张）——仅行动牌                 │
└────────────────────────────────────────────────┘
                    ↓
┌─ 自由行动阶段（不限时 · 玩家手动推进）──────────┐
│ 本回合所有 Agent 博弈围绕"关键事件"展开          │
│                                                 │
│  ┌─ 玩家──┐  ┌─ 三君主 Agent（≤2 轮/对）─────┐│
│  │ 赶路    │  │ 基于关键事件 + 自身利益博弈       ││
│  │ 打牌    │  │ 完成后→"召见"或"决策已定"        ││
│  │ 面谈    │  └─────────────────────────────────┘│
│  └────────┘                                     │
│  打牌成功/面谈完成 → 获得情报牌                   │
└────────────────────────────────────────────────┘
                    ↓
        玩家点击 [进入下回合]
                    ↓
              数值结算 → 即死检查
              → 进入下一回合 / 终局
```

> 详见 [03 · 数值系统](03-数值系统.md)、[04 · Agent 架构](04-Agent架构.md)、[06 · 自由行动阶段详设](06-RTS阶段详设.md)、[12 · 总体剧本](12-总体剧本.md)。

## 一、回合结构

**共 6 回合，每回合 2 个月，总计 1 年。**

| 回合 | 月份 | 秦崛起阶段 | 关键事件主题 | 立场主题题 |
|---|---|---|---|---|
| 第 1 回合 | 1–2 月 | 初露锋芒 | 秦使探虚实 | Q1（合纵/中立/亲秦） |
| 第 2 回合 | 3–4 月 | 施加压力 | 张仪入齐许城 | Q2（合纵/中立/亲秦） |
| 第 3 回合 | 5–6 月 | 兵锋东指 | 合纵之议成形 | Q3（合纵/中立/亲秦） |
| 第 4 回合 | 7–8 月 | 摊牌 | 合纵签字 vs 秦军压境 | Q4（合纵/中立/亲秦） |
| 第 5 回合 | 9–10 月 | 收束 | 决战前夕 | Q5（合纵/中立/亲秦） |
| 第 6 回合 | 11–12 月 | 终局 | 函谷关最后一搏 | Q6（合纵/中立/亲秦） |

> 每回合立场题从 6 题题库中按回合顺序抽取 1 题。详见 [08 · 结局设计 §二 立场判定系统](08-结局设计.md)。

### 阶段零：三立场问卷（回合开始时 · 必答）

每回合开始时弹出 1 题简短选择题，**玩家必须答完才能进入本回合**。

- 题量：每回合 1 题，6 回合共 6 题。
- 题目内容：从 6 题题库中按回合顺序抽取（Q1–Q6）。
- 三选一：A=合纵派（推合纵）/ B=中立自保派（为自己谋利）/ C=亲秦派（推连横帮秦）。
- 跳过机制：若玩家 5 秒不点 → 默认勾选 B（中立自保），避免阻塞。
- 累积识别：6 回合答题后，仲裁器根据累计得分判定玩家立场（合纵派 / 中立自保派 / 亲秦派）。**最终评语按立场 × 终局类型匹配**，共 9 条。

### 阶段一：关键事件生成（回合开始时）

- 仲裁器读取上回合结束时的各国三维 + 玩家三维。
- 结合主线脊骨（秦在哪个阶段）+ 上回合 Agent 博弈关键结果。
- 生成一条 ≤ 60 字的关键事件描述。
- 关键事件设定了"本回合的核心戏剧问题"。
- 详细模板与变体见 [12 · 总体剧本](12-总体剧本.md)。

### 阶段二：抽行动牌

- 从牌堆补行动牌至 5 张。
- **不抽情报牌**。关键事件本身是唯一的公开信息。

### 阶段三：自由行动阶段（不限时）

所有 Agent 博弈围绕**本回合关键事件**展开。玩家赶路、打牌、召见面谈。详见 [06](06-RTS阶段详设.md)。

### 阶段四：回合结束

玩家点击 **[进入下回合]** → 数值结算 → 即死检查 → 下一回合。

## 二、关键事件如何影响 Agent 博弈

关键事件不是写死的剧本，而是**设定博弈的框架**：

- 秦 Agent 的博弈动作优先围绕关键事件（如关键事件是"合纵之议"，秦优先选离间/利诱）。
- 赵/齐 Agent 的博弈动作以关键事件为起点，生成各自的态度和行动。
- 君主的召见困局从关键事件出发——"在这个局面下，纵横家觉得孤该怎么办？"

详见 [07 · 主线与事件涌现](07-主线与事件涌现.md) 与 [12 · 总体剧本](12-总体剧本.md)。

## 三、打牌（二值判定：成功 / 失败 · 打牌前选方向）

自由行动阶段中任何时候可从手牌打出行动牌。判定简化为**成功/失败**两种结果。

### 3.1 成功率公式

| 牌 | 基础成功率 | 加成 |
|---|---|---|
| 游说 | 40% | + 名望 × 0.3 |
| 离间 | 35% | + 心计 × 0.3 |
| 刺探 | 50% | + 心计 × 0.25 |
| 许诺 | 45% | + 名望 × 0.3 |
| 传信 | 40% | + 合纵 × 0.3 |

成功率 = clamp(基础 + 加成 + **消耗情报牌数 × 5%**, 5%, 95%)。打牌时明示。

> **情报牌加成（v7.3.3）**：打牌前可消耗任意数量情报牌，每张 +5% 成功率（受 95% 封顶）。详见 [05 · 卡牌设计 §五](05-卡牌设计.md) 与 §五 情报牌。

### 3.2 打牌前选方向（游说/传信/许诺必须选，刺探/离间不选）

| 牌 | 选项 | 影响 |
|---|---|---|
| **游说** | 推合纵 | 合纵 +8 名望 +3 心计 +3 |
| | 推亲秦 | 合纵 −8 名望 +3 心计 +3 |
| | 中立建言 | 名望 +5 心计 +3 |
| **传信** | 利好合纵的消息 | 合纵 +5 心计 +3 |
| | 利好连横的消息 | 合纵 −5 心计 +3 |
| | 中立情报 | 心计 +5 |
| **许诺** | 承诺援助 | 心计 +6 名望 +3 |
| | 承诺结盟 | 心计 +4 名望 +5 合纵 +3 |
| | 承诺中立 | 心计 +5 名望 +2 |
| **离间** | （不选方向） | 离间 A 与 B 间的信任 |
| **刺探** | （不选方向） | 获取目标君主当前意图 |

> **设计意图**：方向选择机制让玩家**可偏秦可偏合纵可中立**，数值变动根据方向不同——这是"中立化玩家"的核心机制。详见 [03 · 数值系统 §1.1](03-数值系统.md)。

### 3.3 二值结果

| 结果 | 效果 |
|---|---|
| **成功** | 行动生效，数值按选定的方向+牌型变动，获得 1 张情报牌 |
| **失败** | 行动无效，名望 −3（通用惩罚）；不获得情报牌 |

> 取消五档倍率。成功就是成功，失败就是失败。

### 3.4 召见面谈（v7.3.5 · 无成功率 · 立场影响国家三维）

召见面谈**不是二值判定行动**——无成功率、无掷骰、无采纳/拒绝、**不影响玩家三维**。君主提出一个**行动意图**征询玩家，玩家**赞同**则该意图对应的**国家三维**变动生效，**反对**则搁置。详见 §四。

## 四、召见与打字对话（v7.3.5 · 立场影响国家三维 · 无成功率）

抵达召见国 → 君主 Agent **先简述自己正在与其他君王博弈的内容**，**提出一个具体行动意图**（= 其某个博弈动作），最后以**一句简短问句**征询玩家意见 → 玩家输入**一段简短 prompt** 表达赞同/反对/中立 → 按玩家态度结算**国家三维**（不影响玩家三维）。

### 4.1 面谈交互流程

1. **君主开场**：Agent 基于本回合关键事件 + 该国博弈结果，生成一段话——简述"我正在与谁博弈、遇到什么"（≤ 80 字），**提出一个行动意图**，最后一句问句征询玩家（如"寡人有意实施强制措施，足下以为如何？"）。
2. **玩家表态**：输入一段**简短 prompt**（推荐 ≤ 80 字），表达赞同 / 反对 / 中立。不限时。
3. **君主回应 + 结算**：Agent 判读玩家态度 →
   - **赞同** → 君主提议的行动意图对应的**国家三维**变动立即生效（见 [03 · §2.2](03-数值系统.md)）。
   - **反对** → 该行动搁置不生效，记入君主记忆。
   - **中立** → 君主按自身倾向自决（默认执行）。
4. **面谈结束**：该国状态 → 已处理 + 玩家获得 1 张情报牌（面谈摘要）。

> **示例**：秦王"齐王始终不肯归顺，寡人有意实施进一步的强制措施，你觉得如何？" → 玩家肯定 → 秦国威 +、战心 +（军事施压/备战蓄力）。
> **无成功率、不影响玩家三维**：面谈不掷骰、不判成败、不改玩家三维（无 1.5 倍、无名望 −5），因此情报牌加成不适用于面谈。玩家的影响力体现在**国家三维（全局数据）**的走向。
> **LLM 输出 schema**：`{opening, proposed_action, player_stance, response, resolved}`，见 [11b · §四](11b-角色设计稿-程序.md)。

### 4.2 玩家可达的"决策已定"国家

若某国博弈完成但状态为"决策已定"（君主已有定见）：
- 玩家仍可前往（不弹面谈）。
- 抵达时显示该国决策摘要（"齐王决定待价而沽，闭门观望"）。
- 玩家可对该国打牌 → 视为挑战其决策 → 若打牌成功 → 推翻决策（该国状态 → 召见）；若失败 → 名望 −3。
- 玩家不可对"决策已定"国家发起打字面谈（无困局可解）。

## 五、情报牌获取与使用

### 5.1 获取途径

| 途径 | 获得 | 内容 |
|---|---|---|
| 刺探成功 | 1 张 | 目标君主当前的博弈意图 + 对各方态度 |
| 游说/离间/许诺/传信成功 | 1 张 | 该行动的连锁影响 + 目标君主反应摘要 |
| 召见面谈完成 | 1 张 | 面谈结果 + 局势变化摘要 |
| Agent 博弈完成 | 事件流中显示摘要 | 不进入手牌情报牌区，但内容保留在事件流中可回顾 |

### 5.2 情报牌使用（v7.3.3 · 可消耗加成道具）

情报牌是**双重用途道具**：

- **信息认知**：随时回顾，了解各君主意图与局势。
- **成功率加成**：打主动行动牌（游说/离间/刺探/许诺/传信）前，可消耗**任意数量**情报牌——**每张 +5% 成功率**，无硬上限，受 `clamp(5%, 95%)` 封顶。消耗后移出情报牌区（不可复用）。

> **取舍**：情报牌"留着看信息" vs "花掉冲高成功率"形成核心策略张力。详见 [05 · 卡牌设计 §五](05-卡牌设计.md)。

## 六、回合推进

[进入下回合] 按钮规则：
- 灰色：仍有 Agent 博弈未完成（某对仍在第 1/2 轮）→ "还有国家在谈判中"
- 亮起：所有三对博弈已完成（全部进入"召见"或"决策已定"）→ 可推进
- 亮起 + 有未处理召见：提示"还有召见未处理，确定进入下回合？"→ 玩家可选"再等等"或"确定进入"

## 七、典型一局时间线

```
第 1 回合（1–2 月）
  问卷：Q1（识别你的初始立场）
  关键事件："秦使分赴各国——大棋局正在展开"
  玩家：刺探各国，在赵建立关系，尝试推合纵起步
  → 获得 2–3 张情报牌

第 2 回合（3–4 月）
  问卷：Q2（识别你的行动风格）
  关键事件："秦拔宜阳，张仪入齐许诺割城"
  玩家：赶往齐国，阻止齐被秦拉拢
  → 关键决策回合

第 3 回合（5–6 月）
  问卷：Q3
  关键事件："秦兵锋东指，赵齐各自计算"
  玩家：在赵和齐之间斡旋

第 4 回合（7–8 月）
  问卷：Q4
  关键事件（状态分支）：
    - 若合纵将成 → "六国使节齐聚邯郸——签字时刻"
    - 若秦占优 → "秦军压境，各国噤声"
    - 若僵持 → "合纵与连横并立——纵横家站哪边"
  玩家：全力推动自己的目标方向

第 5 回合（9–10 月）
  问卷：Q5
  关键事件：局势收束
  玩家：最后的斡旋

第 6 回合（11–12 月）
  问卷：Q6
  关键事件："秦军陈兵函谷关外——最后一搏"
  玩家：终局前最后机会 → 终局判定 → 立场评语
```

## 八、MVP 核心验证点

1. **关键事件是否提供了足够的戏剧聚焦？** → 每回合有明确的核心矛盾。
2. **二值判定是否让反馈更清晰？** → 成功/失败二元，玩家立刻知道结果。
3. **方向选择机制是否让"可偏秦"成为现实？** → 玩家打游说/传信/许诺牌可选方向，数值变动按方向走。
4. **面谈立场表达是否自然？** → 君主简述商议 + 一句问句 → 玩家简短表态 → 立场被听取并影响君主后续决策（无成功率判定）。
5. **三立场问卷是否提供"被识别"的体验？** → 6 回合 6 题累积出 3 立场之一，玩家看到自己的立场判定。
6. **6 回合节奏是否紧凑？** → 比 12 回合更快到达戏剧高潮。
7. **再来一局的意愿** → 不同关键事件推进方向 + 不同立场评语是否产生复玩动机。
