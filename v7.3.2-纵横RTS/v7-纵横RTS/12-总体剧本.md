# 12 · 总体剧本（v7.3.1 · 6 回合 × 2 月 = 12 个月 · 关键事件详写）

> **目的**：为每回合的"关键事件"提供**详细剧本**，供 Agent 调用推演。策划 D2 完成初稿，D3 调试。
> **结构**：每回合包含「月份-事件主题-关键事件描述-本回合 Agent 议题-玩家可用卡牌组合提示-立场问卷题-状态分支」。
> **调用接口**：仲裁器在回合开始时，根据上回合数值状态 → 匹配本回合模板 → 注入 Agent prompt。
> **接口代码**：
> ```python
> def get_key_event(round_num: int, prev_state: dict) -> str:
>     state = classify_state(prev_state)
>     template = match_template(round_num, state)
>     return template["event"]
> ```

## 第 1 回合 · 1–2 月 · 初露锋芒

### 月份 1（正月）

**事件主题**：秦使入赵
**关键事件描述**：
> 秦王遣内史腾携国书入赵，以"互市"为名探虚实。赵廷震动——这是秦近 5 年来首次主动遣使。平原君请赵王"以礼相待、暗观其意"，廉颇请"扣而问之"。

**本回合 Agent 议题**：
- 秦：试探赵的虚实 + 评估赵齐关系。
- 赵：要不要接见秦使？接见后谈什么？
- 齐：观望——这是秦赵的事，与我何干。

**玩家可用卡牌组合提示**：
- 游说（推合纵）打赵：建议赵王警惕秦使，可能触发赵的"求盟联齐"动作。
- 刺探秦：了解秦的真正意图。
- 传信（利好连横）打齐：提醒齐王注意秦的动向（间接推动齐关注）。

**立场问卷题**：Q1（合纵/中立自保/亲秦）。
**状态分支**：固定 1 条（无前序状态）。

**本回合 Agent 议题**：

- 赵：自保 + 迫切求援。

**状态分支**：

**玩家可用卡牌组合提示**：

- 游说（推合纵）打齐：趁齐担心战火，推齐参与合纵。

**立场问卷题**：Q2（合纵/中立自保/亲秦）。

**本回合 Agent 议题**：

**状态分支**：

### 状态判断逻辑

```python
def classify_state(prev_state: dict) -> str:
    # 第 1 回合：无前序
    if prev_state["round"] == 0:
        return "init"

    # 第 2/3 回合：赵齐盟信和分支
    if prev_state["round"] in [1, 2]:
        mengxin_sum = prev_state["zhao_mengxin"] + prev_state["qi_mengxin"]
        if mengxin_sum < 70:
            return "hezhong_blocked"
        else:
            return "hezhong_stalemate"

    # 第 4 回合：摊牌分支
    if prev_state["round"] == 3:
        avg_mengxin = (prev_state["zhao_mengxin"] + prev_state["qi_mengxin"]) / 2
        if avg_mengxin >= 60:
            return "hezhong_signing"
        elif prev_state["qin_guowei"] >= 80:
            return "qin_dominant"
        else:
            return "stalemate"

    # 第 5 回合：收束分支（接续回合 4）
    if prev_state["round"] == 4:
        # 接续回合 4 的状态
        ...

    return "default"
```

```python
def inject_key_event(agent_prompt: str, key_event: dict) -> str:
    injection = f"""

# 本回合关键事件
**{key_event['title']}**

{key_event['description']}

# 你应该如何应对
{key_event['agent_topics'][agent_name]}

"""
```
