# 01 · 游戏概述与定位（v7.3.1 · 3.5 天）

## 一句话

你是战国的一位纵横家。每回合开局先答一个简短问题（系统用这些来识别你的立场），随后抽行动牌，进入**自由行动阶段**——一张大地图上，你赶路奔波于秦赵齐三国之间，与此同时三位君主 Agent 在同步密谈、施压、结盟与背叛。当你抵达某国，君主或许已因自主博弈而陷入困局，在"**召见**"中等你用**自己的话**来游说——**不限时**，想多久想多久。**情资不是白给的，你得靠行动和面谈自己挣**。**你不只是在选择选项——你是在告诉系统你是谁，然后在真实的、可以看到 AI 之间博弈的世界中，一步步撬动天下大局。**

## 核心体验（五根支柱）

1. **世界在你眼前实时博弈**：自由行动阶段中，各国君主 Agent 同步进行谈判、施压、密谋。地图上使节移动、国家状态切换、数值实时变化——你**看到** AI 在博弈，而不是凭空脑补。

2. **赶路本身就是策略**：Agent 最多博弈 2 轮就会出结果——你要在谁谈判时去打断？谁召见后去面谈？选择先见谁、什么时候出发，本身就是纵横术。

3. **你的话就是武器**：抵达召见中的国家后，君主的困局摆在面前——你**打字**回应，**不限时**。没有选项列表，你说的话直接影响君主 Agent 的裁决。

4. **情报靠行动挣**：回合开始不抽情报牌。刺探、打牌成功、完成面谈——这些才会给你情报牌。你越主动，信息越多。

5. **你的立场由自己回答**：每回合开始弹出 1 个简短问题，你勾选最像自己的选项——这些选择累加起来，构成一个**三立场画像**（合纵派 / 中立自保派 / 亲秦派），最终决定你的结局评语怎么写。**没有"正确答案"，系统只关心你是谁**。

## 这是什么

- **《苏丹的游戏》的卡牌策划 × 召见打字深度交互 × 不限时的自由博弈 × 三立场玩家画像**：每回合先回答一个问题（识别你的立场），再抽牌布局，自由行动阶段中赶路、打牌、面谈，直到你决定推进到下一个月。
- **一台 15–20 分钟的动态棋局**：地图 + 手牌 + 三维即死 + AI 之间可见的博弈 + 涌现终局 + 玩家画像。
- **群体智能的真正展示**：玩家能**看到** AI 君主之间在做什么——地图上、事件流中、国家数值变化里。
- **三立场识别的真正展示**：每回合问你 1 个问题，6 回合累积成 3 立场之一——你不是选剧本，你是告诉系统你是谁。

## 不是什么

- 不是纯卡牌游戏（卡牌是行动工具，核心交互在赶路+召见+打字）。
- 不是选项分支（召见时玩家**打字**，无预设选项）。
- 不是限时 RTS（回合不限时，面谈不限时，玩家自主推进）。
- 不是情报白给（情报牌通过行动获得，不是开局抽）。
- 不是预设立场（系统不假设你帮秦还是帮赵——你的立场由 3 立场问卷定义，由你的回答累加而成）。

## 本期范围（硬约束）

- **玩家**：1 位纵横家，不臣属任何君主。
- **Agent**：3 个 LLM 君主 Agent + 每个 1–2 个规则近臣。Agent 在自由行动阶段**持续并行运作**，但**每对博弈上限 2 轮**。
- **回合**：至多 **6 大回合**（12 个月 / 1 年）。每大回合 = 三立场问卷 1 题 → 抽行动牌 → 自由行动阶段（不限时，玩家点击"进入下回合"推进）。
- **交互**：每回合问卷 + 卡牌策划 + 地图赶路 + 打字面谈。**不做选项分支**。面谈不限时。
- **情报获取**：回合初只抽行动牌；情报牌通过打牌成功/召见完成/刺探成功获得。
- **团队 / 周期**：策划 + 程序 + 美术，**3.5 天**（含 0.5 天集成打磨）。
- **形态 / 引擎**：文字 + 地图 + 卡牌 UI + 打字输入框 + 问卷弹窗；新建 Godot 项目。

## 设计铁律

1. **自由行动阶段 Agent 同步博弈**：玩家移动/打牌/面谈的同时，君主 Agent 持续进行自主博弈。每对博弈 ≤ 2 轮，完成后进入"召见"或"决策已定"。不得退化为"玩家回合→AI 回合"。

2. **数值即死不可豁免**：玩家三维任一归零或爆表 → 立即死亡。无续命。

3. **打字替代选项，不限时**：召见面谈时玩家必须打字。不做选项列表。君主提出行动意图，Agent 读取玩家真实文本判读赞同/反对/中立 → 影响国家三维（v7.3.5 · 无成功率）。玩家可以慢慢想、慢慢打。

4. **卡牌方向选择**：打牌前玩家必须选择行动方向（推合纵/推亲秦/中立信息），方向不同→数值变动方向不同。这让玩家"可偏秦"不是空话。

5. **情报通过行动获得**：回合初不抽情报牌。刺探卡牌、打牌成功、完成召见面谈——这些行为产出情报牌。不行动 = 两眼一抹黑。

6. **三立场问卷每回合弹出**：每回合开始 1 题（题库 6 题），3 选 1 测立场——**合纵派 / 中立自保派 / 亲秦派**。**题目是预制的，识别是系统的，玩家不操心"测什么"**。

7. **博弈全程可见**：地图上的使节移动、国家状态变化、数值实时更新、事件流滚动——玩家看到 AI 之间的博弈动态。

8. **降级链常驻**：LLM 断连 → 全局 mock 规则博弈，地图上的状态切换/数值变化照常运转。

## 与上级的对话

- livewang 想验证"AI 自主博弈涌现"。本方案：3 君主 Agent 同步博弈（最多 2 轮/对）→ 涌现出来的态势（合纵/破盟/僵持）通过态势终局判定。
- 玩家是纵横家，不掌兵不掌国，武器是信息和说服力。
- 玩家画像不再用终局问卷"事后审问"——而是每回合 1 题，渐进式识别，6 回合累积出三立场。
- 5 天→3.5 天的砍法：mock 优先、UI 简化、可视化用占位、态度问卷改每回合分散、评语从 48 条减为 9 条。
