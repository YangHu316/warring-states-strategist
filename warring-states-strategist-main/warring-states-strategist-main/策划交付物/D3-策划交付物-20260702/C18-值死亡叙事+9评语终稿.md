
## 五、终局判定流程

```
第 6 回合结束
  ├─ 三维值死亡检查（优先）→ 0/100 → 值死亡叙事（无评语）
  └─ 未死
      ├─ 读取各国三维 → 判定终局类型（合纵却秦 / 连横破盟 / 纵横未决）
      ├─ 读取玩家 6 题立场得分（C06）→ 判定玩家立场
      ├─ 匹配评语（3 终局 × 3 立场 = 9 条）→ 出评语 + 立场标题
      └─ 弹落幕屏（bg_ending_{ending_type}.png + 评语 + 标题）
```
