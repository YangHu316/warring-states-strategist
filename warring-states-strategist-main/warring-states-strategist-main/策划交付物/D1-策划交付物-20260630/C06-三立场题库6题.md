## 一、设计原则（08 §一 · 锁定）

- **问卷只测立场**，不测人格。玩家不会被"测出 ESTJ"这种东西。
- **3 立场对应 3 套剧情反馈**。
- **中立不模糊**——中立自保派是清晰的"为自己"，不是"两个都沾"。

## 二、6 题题库（终稿 · 每回合 1 题）

### 回合 1 · Q1（初探）

> **场景提示**：「列国君主邀请你入幕。你会？」

| 选项 | 文本 | 立场信号 |
|---|---|---|
| **A** | 接受赵国——合纵若成，赵是盟主 | **合纵 +1** |
| **B** | 看谁出价更高——纵横家不站队 | **中立 +1** |
| **C** | 接受秦国——得时则驾，不得则隐 | **亲秦 +1** |

> **20 秒未点 → 默认 B（中立自保）**（避免阻塞）

### 回合 2 · Q2（施压）

| **A** | 帮赵——合纵是抗秦正道 | **合纵 +1** |

| **C** | 给秦递消息——识时务者为俊杰 | **亲秦 +1** |

### 回合 3 · Q3（对峙 · v7.3.7 P1-20 措辞明确化）

> **场景提示**：「平原君为合纵奔走，廉颇主张独战。他们都有道理，但二选一。你会？」

| 选项 | 文本 | 立场信号 |
|---|---|---|
| **A** | 选平原君——合纵才是抗秦正道 | **合纵 +1** |
| **B** | 不选——让他们自己吵 | **中立 +1** |
| **C** | 选廉颇——独战虽险，至少赵国命运由赵人自己定 | **亲秦 +1** |

> **v7.3.7 P1-20 措辞明确化（✅ 已拍板采用方案3）**：
> **问题**：廉颇是赵国主战派（不亲秦），但其逻辑"独战不依赖齐 → 给秦可乘之机"被解读为亲秦路径，玩家反馈"措辞不顺"。试玩 6/10 玩家选错。
> **已采纳方案3**（v7.3.7 P1-20 用户拍板）："选廉颇——独战虽险，至少赵国命运由赵人自己定。"
> **理由**：明示"赵国命运由赵人自己定"，避免歧义；与 C23 §五 建议 3 一致。
> **状态**：✅ **已锁定**（2026-07-02 拍板）。

## 三、计分与判定规则（08 §2.3 · 锁定）

### 3.1 立场得分

| 立场 | 6 题累计得分 |
|---|---|
| **合纵派** Hézòng | 0–6 分 |
| **中立自保派** Zhōnglì | 0–6 分 |
| **亲秦派** Qīnqín | 0–6 分 |

### 3.2 判定规则（取最高分，平分时按下表打破）

| 情况 | 判定结果 |
|---|---|
| 合纵分 > 中立分 且 合纵分 > 亲秦分 | **合纵派** |
| 亲秦分 > 中立分 且 亲秦分 > 合纵分 | **亲秦派** |
| 中立分最高 | **中立自保派** |
| 三方同分（极少） | 中立自保派（默认稳健） |
| 合纵=亲秦 > 中立 | 看哪一方**第 6 回合题**得分 → 决出 |
| 合纵=中立 > 亲秦 | **中立自保派**（稳健优先） |
| 亲秦=中立 > 合纵 | **中立自保派** |

> **简化原则**：中立自保派是默认兜底——玩家没表态明确就归这里。

### 3.3 UI 流程（08 §三 · 锁定）

```
回合开始
  ├─ 弹出关键事件横幅（顶部）
  ├─ 弹出本回合问卷 modal（1 题）
  │   ├─ 玩家点击 A/B/C → 计分
  │   └─ 20 秒未点击 → 默认 B（中立自保）→ 计分
  ├─ 关闭问卷 → 进入"抽行动牌 + 自由行动"阶段
  └─ 状态栏显示"本回合立场已记录"
```

> **问卷 UI 简化为 3 个大按钮**（A/B/C）+ 问题文字。
> **不显示"立场测试"字样**——避免引导玩家作弊。
