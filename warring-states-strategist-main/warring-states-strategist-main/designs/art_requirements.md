# 战国谋士 — D1 美术需求

## 1. 美术风格

- **总体风格**：像素艺术（Pixel Art），战国时代东方水墨意境与像素融合
- **主调色板**（hex）：
  - 主色：`#2C1810`（墨黑）、`#8B4513`（赭石）、`#D4A574`（土黄）
  - 强调色：`#C73E3A`（朱红/秦）、`#3A5F8E`（深蓝/赵）、`#5B8C3E`（青绿/齐）
  - 背景色：`#F4E4BC`（宣纸黄）、`#1A1410`（深背景）
- **参考意境**：战国竹简、青铜器纹饰、水墨写意
- **D1 范围说明**：D1 仅生成最少必需的占位资产（背景 + 3 君主头像占位 + 1 卡牌底图占位），完整美术资产 D2+ 再补
