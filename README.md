# 战国谋士 · Warring States Strategist

一个 3.5 天 GameJam 项目：**战国纵横家 × AI 群体智能 × 三立场玩家画像**。

你扮演一位战国纵横家，穿梭于秦、赵、齐三国之间。三位君主 Agent 由 LLM 驱动，在你眼前实时博弈；你用打字（不限时）撬动他们的决策。6 回合走完，你的立场（合纵派 / 中立自保派 / 亲秦派）会决定评语。

## 核心机制

- **同步博弈**：秦/赵/齐 三个 LLM Agent 每回合最多 2 轮独立决策，围绕关键事件展开
- **卡牌方向**：游说/传信/许诺打前必选方向（推合纵/推亲秦/中立），数值变动跟着方向走
- **打字面谈**：抵达召见国 → 打字回应君主困局 → LLM 二值裁决（采纳/拒绝）
- **玩家立场**：每回合 1 题三选一，6 题累积成合纵派/中立自保派/亲秦派
- **降级链**：LLM 断线全部落回 mock 决策表 + 关键词面谈

## 技术栈

- Godot 4.6.3（GL Compatibility）
- DeepSeek API（`deepseek-v4-flash` Agent 决策，`deepseek-v4-pro` 面谈裁决）
- 三立场 × 三终局 = 9 条评语，MBTI 已废弃

## 运行

1. Godot 4.6.3 打开 `project.godot`
2. 首次启动会在 `user://config.cfg` 生成配置模板，把你的 DeepSeek `api_key` 填进去（不填则全走 mock）
3. F5 运行 `scenes/main.tscn`
