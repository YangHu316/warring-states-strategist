## 1. 背景与动机

反向检测报告共识别 13 个 P0 问题，其中 5 个属于"数值曲线类"，均触及 `AI_INTEGRATION_GUIDE.md` §7.5 不可改项清单，需走 RFC 流程。

**核心矛盾**（来自反向检测报告 §五 3 次推演）：
- 推合纵路径 → 30 张 = 即死
- 推亲秦路径 → 5 张 = 即死
- **中立自保**是当前唯一可通关路径

这与游戏"策略选择 + 阵营对抗"的定位严重不符：玩家无论选哪一阵营都是速败。

## 4. 详细修改方案

### 4.1 P0-5：终局 A 阈值修订

**现状**
```
终局 A（合纵却秦）= 赵盟信 ≥ 55  AND  齐盟信 ≥ 55  AND  秦国威 ≤ 75
```

**问题推演**
- 齐盟信初始 45，drift 速率约 -2/轮（init 观望渔利）
- 6 轮后齐盟信预计：45 + (-2 × 6) + (玩家游说齐 +5) = 28 左右
- 即便每轮都游说齐（仅 3 张 游说牌可用），齐盟信约 33，仍 < 55
- **结论：终局 A 不可达**

**修订方案 A（推荐）：调整阈值**
```
终局 A（合纵却秦）= 赵盟信 ≥ 45  AND  齐盟信 ≥ 35  AND  秦国威 ≤ 80
```

**修订方案 B（备选）：调整 drift**
```
齐 mock 观望渔利分支：齐盟信 -4 → 0
齐 mock 主动合纵分支：齐盟信 +5 → +8
```

**推荐采用方案 A**（影响面小、推演可验证）。

---

### 4.2 P0-6：推亲秦曲线修订

**现状**
```
推亲秦 1 张：名望 -6 / 心计 -3
推亲秦累计：名望 -30（5 张后触发名望 20 君主猜疑）
```

**问题推演**
- 玩家名望 50 起步
- 推亲秦 1 张：名望 44（+ 君主猜疑 4）
- 推亲秦 5 张：名望 20，触发"名高震主"或"君主疑为秦间"即死
- **结论：5 张 = 必死**

**修订方案**
```
推亲秦 1 张：名望 -6 → 名望 -3
推亲秦累计阈值：名望 20 → 名望 10
玩家名望初始：50 → 60
```

**推演验证（修订后）**
- 推亲秦 5 张：名望 60 - 3×5 = 45（安全）
- 推亲秦 10 张：名望 30（接近警戒线）
- 推亲秦 15 张：名望 15（危险）

---

### 4.3 P0-7：推合纵曲线修订

**现状**
```
推合纵 1 张：心计 -2 / 名望 +3
推合纵累计：心计 -60（30 张后心计 0 即死）
```

**问题推演**
- 玩家心计 40 起步
- 推合纵 1 张：心计 38
- 推合纵 30 张：心计 0，即死
- **结论：30 张 = 必死**

**修订方案**
```
推合纵 1 张：心计 -2 → 心计 -1
玩家心计初始：40 → 50
推合纵 30 张：心计 50 - 30 = 20（安全）
推合纵 50 张：心计 0（极难达到，符合"持续操作上限"）
```

---

### 4.4 P0-8：齐王 init 分支修订

**现状**
```
C11 mock 决策表：
  init 分支（前 2 轮）：
    → 观望渔利
    → 齐盟信 -4 / 轮
```

**问题推演**
- 齐盟信初始 45
- 2 轮 init 后：45 - 8 = 37
- 后续即便走合纵分支：37 + 5×4 = 57（可能）
- **结论：修订后 4 轮内可达 55**

**修订方案**
```
齐王 init 分支（前 2 轮）：
  轮 1：观望渔利（齐盟信 -2 而非 -4）
  轮 2：随机触发，50% 观望渔利 / 50% 主动合纵
```

**备选方案**
```
齐王 init 分支（前 2 轮）：
  → 全部改为"主动合纵观望"（齐盟信 +0，即不增不减）
  → 避免开局必跌
```

**推荐采用备选方案**（更稳健，避免随机性）。

---

### 4.5 P0-9：玩家/国家三维机制对称化

**现状**
```
玩家三维（合纵/名望/心计）：
  - 0~100 clamp
  - 0 或 100 触发即死

国家三维（国威/盟信/战心）：
  - 无 clamp（可超出 0~100）
  - 无即死（仅终局判定）
```

**问题**
- 玩家一不留神就死，国家可放心大胆玩
- 体验割裂，玩家紧张感 vs AI 自由度的反差过大

**修订方案 A（推荐）：国家三维加 clamp 不加即死**
```
国家三维：
  - 0~100 clamp（与玩家对称）
  - 无即死（保持 AI 自由度）
  - 终局判定不变
```

**修订方案 B：玩家三维放宽到无即死**
```
玩家三维：
  - 0~100 clamp 保留
  - 取消即死，仅触发对应"叙事事件"（如 名望 0 = 被放逐但继续游戏）
```

**推荐采用方案 A**（更符合"对称感"且改动小）。

| 风险 | 等级 | 缓解措施 |

| 阈值放宽后终局 A 过于容易 | 中 | 推演验证：确保 3 路径各有可行通关方案 |
