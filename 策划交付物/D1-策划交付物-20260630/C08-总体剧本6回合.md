## 二、6 回合关键事件初稿（每条 100–200 字）

### 第 1 回合 · 初露锋芒

**事件主题**：秦使分赴各国——大棋局正在展开

> 秦王遣内史腾携国书入赵，以"互市"为名探虚实——这是秦近 5 年来首次主动遣使。赵廷震动，平原君请赵王"以礼相待、暗观其意"，廉颇请"扣而问之"。与此同时，张仪未至但秦的国书已先一步送到临淄，提议"齐秦互不侵犯"，并许以三城为"诚意"。齐王笑纳国书但不置可否。**秦使分赴各国——大棋局正在展开。**

**本回合 Agent 议题**：
- 秦：试探赵的虚实 + 评估赵齐关系 + 留下"连横伏笔"。
- 赵：要不要接见秦使？接见后谈什么？听闻秦使入齐后如何应对？
- 齐：渔利派得意——这是齐国的好机会，让秦赵先斗。

**状态分支**：固定 1 条（无前序状态）。
**立场问卷**：Q1（合纵/中立自保/亲秦）。

---

### 第 2 回合 · 施加压力

**事件主题**：秦拔宜阳，张仪入齐许诺割城

> 秦军拔宜阳，斩首三万。韩王上表请降，宜阳成秦东进的前哨。函谷关外战云密布——赵边境压力陡增。秦拔宜阳之威未歇，张仪亲至临淄，以连横之说游说齐王，携"三城即日奉上"的重礼。齐王宴请之，孟尝君在侧不发一言；宴罢，齐王赐张仪金帛，遣之归秦——但暗中让孟尝君"且观成败"。**赵王听闻此事，恨齐"私通秦"。**

**本回合 Agent 议题**：
- 秦：趁势加压 + 离间赵齐 + 连横初见成效（齐王收了金帛）。
- 赵：自保 + 迫切求援。
- 齐：继续观望——但开始担心战火烧到东边。

**状态分支**：
- 若上回合赵齐盟信和 < 70 → 模板 A："秦拔宜阳，张仪携重礼入齐——'齐王若观望，三城即日奉上。'"
- 若上回合赵齐盟信和 ≥ 70 → 模板 B："秦急遣使分赴赵齐——'谁先动，秦就先打谁。'离间之计，赤裸裸摆在台面上。"

**立场问卷**：Q2（合纵/中立自保/亲秦）。

---

### 第 3 回合 · 兵锋东指

**事件主题**：合纵之议成形 vs 秦的离间试探

> 赵国平原君奔走于列国之间，倡合纵之说。楚、魏观望，齐王不置可否。赵王急召纵横家——"齐王说让我先动，可孤若先动，秦必攻赵。"秦相魏冉率兵压赵境，声言"若赵不合纵，秦愿与赵分河为界"——这是秦的离间计：让赵以为秦愿意"半合纵"。但楚、魏闻讯后"以为秦赵和好"，退出合纵之议。**赵被秦的话动摇——"若秦愿分界，何必合纵？"**

**本回合 Agent 议题**：
- 秦：加紧军事调动 + 离间 + 诱赵 + 瓦解合纵。
- 赵：求盟 + 备战 + 犹豫。
- 齐：笑看风云——"让他们自己吵"。

**状态分支**：
- 若上回合赵齐盟信和 < 70 → 模板 A："合纵之议在邯郸成形——秦感受到了威胁，开始加紧军事调动。"
- 若上回合赵齐盟信和 ≥ 70 → 模板 B："六国各怀鬼胎，秦趁虚而入——合纵还没成形，就要散了。"

**立场问卷**：Q3（合纵/中立自保/亲秦）。

---

### 第 4 回合 · 摊牌

**事件主题**：合纵签字 vs 秦军压境

> 六国使节齐聚邯郸——合纵到了签字的关键时刻。但齐王私下仍在和秦通信，赵王收到密报。关键走向有三种可能：若合纵将成，则条款要求各国交出部分兵权，赵王犹豫；若秦占优，则秦军大举压境，某国先降，秦的盟信大涨；若僵持不下，则赵王召见纵横家，希望得到"该不该签"的答案。**这是纵横家入局以来最关键的一个回合。**

**本回合 Agent 议题**：
- 秦：阻止签字 + 加速离间（若 A/B）；伺机而动（若 C）。
- 赵：决定是否签。要在"信齐"和"疑齐"之间做选择。
- 齐：脚踏两船 + 待价而沽。

**状态分支**（关键节点）：
- 若前三回合赵齐盟信均值 ≥ 60 → 模板 A："六国使节齐聚邯郸——合纵到了签字的关键时刻。"
- 若秦国威 ≥ 80 → 模板 B："秦军压境，各国噤声——谁先出头谁先死。"
- 否则 → 模板 C："合纵与连横并立——纵横家站哪边。"

**立场问卷**：Q4（合纵/中立自保/亲秦）。

---

### 第 5 回合 · 收束

**事件主题**：决战前夕 / 临阵变局

> 局势收束：合纵若成 → "六国联军在邯郸集结——函谷关外，秦军亦在集结"；合纵若败 → "秦军推进——某国先降，盟信大涨"；僵持 → "决战前夕——双方都在等对方先动"。临阵变局接踵而至：若合纵成则六国联军出动——但齐王临阵倒戈，与秦暗通；若合纵败则秦军节节推进——但赵王决意独战，誓守邯郸；若僵持则双方在边境对峙——纵横家游走于两国之间。**这是纵横家入局以来最关键的第二个回合——斡旋的最后窗口。**

**本回合 Agent 议题**：
- 秦：根据上回合结果调整策略 + 扩大战果或稳定战线。
- 赵：求援 or 独战。
- 齐：继续渔利 or 倒戈。

**状态分支**：接续回合 4 的状态（详见 §三 状态分支详表 · 回合 5 行）。
**立场问卷**：Q5（合纵/中立自保/亲秦）。

---

### 第 6 回合 · 终局

**事件主题**：函谷关外最后一搏

> 秦军陈兵函谷关外。合纵联军（或独战赵军）与秦军对峙——这是最后一搏。若秦国威占优，则"秦军压境——明日决战"；否则"秦使入赵求和——这是最后的和谈机会"。无论决战还是和谈，纵横家在 6 回合里做的所有选择——推合纵、推亲秦、中立斡旋——都会在这里兑现。**这是终局前的最后机会。**

**本回合 Agent 议题**：
- 秦：决定"和"或"战"。
- 赵：决定"战"或"降"。
- 齐：决定"援"或"弃"。

**状态分支**：
- 若秦国威 ≥ 80 → 模板 A："秦军压境——明日决战。"
- 否则 → 模板 B："秦使入赵求和——这是最后的和谈机会。"

**立场问卷**：Q6（合纵/中立自保/亲秦）。

## 九、待确认项 ⚠️

1. **v7.3.2 单层结构 vs v7.3.1 双层结构的优劣** —— 单层更紧凑，但失去了"回合内时间推进感"。**用户在 D2 试玩后反馈是否需要回退到双层**。
2. **状态分支阈值的"赵齐盟信和"和"秦国威"** 沿用 12 文档数值，未与 C01 §3.1 终局阈值做交叉验证（终局阈值用"赵盟信 ≥ 55 且 齐盟信 ≥ 55 且 秦国威 ≤ 75"，但状态分支用"盟信和"和"秦国威 ≥ 80"）—— **程序与策划需在 D2 共同确认这两个阈值的内部一致性**。
3. **第 5 回合"接续回合 4"的状态判断** 在 12 文档中只给了占位符"..."，**D2 终稿时需具体化**（如"接续回合 4 的 hezhong_signing 状态 → 进入收束分支"）。
4. **第 4 回合 3 模板的优先级**：12 文档说"盟信均值 ≥ 60"或"秦国威 ≥ 80"或"其余"三选一，**用户需确认模板 B（秦国威 ≥ 80）是否要放在模板 A 之前**（更紧急的应该先匹配？）。
