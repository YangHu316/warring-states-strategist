## 二、5 近臣权重表（04 §四 · 程序锁定 · 全部沿用 v7.3）

> 每个近臣在 Agent 决策时作为**倾向权重**注入 prompt（或 mock 决策表的加权逻辑）。

| 君主 | 近臣 | 倾向 | **权重影响（注入 Agent / mock 表）** |
|---|---|---|---|
| 秦王 | 张仪 | 连横第一推手 | 连横利诱权重 ×1.5；备战蓄力 ×0.6 |
| 秦王 | 魏冉 | 穰侯，务实扩张 | 军事施压权重 ×1.4 |
| 赵王 | 平原君 | 主合纵 | 求盟联齐权重 ×1.4；骑墙观望 ×0.5 |
| 赵王 | 廉颇 | 主战不信秦 | 备战固境权重 ×1.3；骑墙观望 ×0.5 |
| 齐王 | 孟尝君 | 谨慎渔利派 | 观望渔利权重 ×1.3；趁火打劫 ×0.5 |

### 2.1 权重使用规则（锁定）

- **注入 Agent prompt**：以"近臣低语"形式注入到"5. 近臣倾向"模块（5–10 字短句）→ 11b §三 已锁定
- **注入 mock 决策表**：作为"加权逻辑"出现在 mock.json 的 `weights` 字段

### 2.2 5 近臣 Prompt 注入片段（11b §三 · 锁定）

#### 张仪（秦）· 连横第一推手

```markdown
# 你的近臣
- **张仪**（连横第一推手）：当你考虑"连横利诱"时，他低声说："齐王贪利，许以三城可定。"
  权重影响：连横利诱动作的 confidence +2。
- **魏冉**（穰侯）：当你犹豫时，他低声说："战机稍纵即逝。"
  权重影响：军事施压动作的 confidence +1。
```

#### 魏冉（秦）· 穰侯 · 务实扩张

```markdown
- **张仪**：当你犹豫"是否用兵"时，他说："三城换一国，何乐不为。"
  权重影响：连横利诱 confidence +1。
- **魏冉**：当你犹豫"是否用兵"时，他说："兵临城下自有答复。"
  权重影响：军事施压 confidence +2。
```

#### 平原君（赵）· 主合纵

```markdown
- **平原君**（主合纵）：当你想联齐时，他说："合纵抗秦是赵国唯一出路。"
  权重影响：求盟联齐 confidence +2。
- **廉颇**（主战不信秦）：当你想联齐时，他说："齐未必真心，独战亦未尝不可。"
  权重影响：求盟联齐 confidence −1，备战固境 confidence +1。
```

#### 廉颇（赵）· 主战不信秦

```markdown
- **平原君**（主合纵）：当你想独战时，他说："合纵可保赵氏宗庙。"
  权重影响：求盟联齐 confidence +1，备战固境 confidence −1。
- **廉颇**（主战不信秦）：当你想独战时，他说："廉颇老矣，尚能一战。"
  权重影响：备战固境 confidence +2，骑墙观望 confidence −1。
```

#### 孟尝君（齐）· 谨慎渔利派

```markdown
- **孟尝君**（谨慎渔利派）：当你想动手时，他说："动则必败，观望方为上策。"
  权重影响：观望渔利 confidence +2，趁火打劫 confidence −1，待价而沽 confidence −1。
```

## 三、5 近臣视觉关键词（11a §二 · 美术锁定）

> 美术在 D1 出草图 / D2 出线稿 / D3 出终版。本节给出统一的视觉规范。

### 3.1 张仪（秦 · 连横第一推手）

| 维度 | 关键词 |
|---|---|
| **职能** | 秦的连横核心谋士 |
| **性格** | 狡黠、口才极佳、唯结果论 |
| **视觉** | 瘦削、修长、眼神闪烁、嘴角永远带笑 |
| **服饰** | 秦黑红深衣，文士冠，**手持一卷竹简**（"连横"二字隐约可见） |
| **姿态** | 身体前倾，像在说服人 |
| **背景** | 秦黑红帷幔，远景栈道 |

### 3.2 魏冉（秦 · 穰侯 · 务实扩张）

| 维度 | 关键词 |
|---|---|
| **职能** | 秦的军事扩张推手 |
| **性格** | 粗犷、果断、重实务、军功至上 |
| **视觉** | 壮实、方脸、络腮胡（细画）、眼神锐利 |
| **服饰** | 秦黑红深衣+**半甲护肩**、武冠 |
| **姿态** | 单手叉腰或**握剑柄** |
| **背景** | 函谷关外军营、战车剪影 |

### 3.3 平原君（赵 · 主合纵）

| 维度 | 关键词 |
|---|---|
| **职能** | 赵的合纵外交推手 |
| **性格** | 儒雅、有大局观、略显书生气 |
| **视觉** | 清瘦、长脸、温和、眼神坚定 |
| **服饰** | 赵青灰深衣、文士冠、**手持玉圭** |
| **姿态** | 端正拱手，像在行礼 |
| **背景** | 赵国书斋、竹简堆 |

### 3.4 廉颇（赵 · 主战不信秦）

| 维度 | 关键词 |
|---|---|
| **职能** | 赵的武将，独战派 |
| **性格** | 刚直、有傲气、老将不服老 |
| **视觉** | 魁梧、白发、浓眉、疤痕（细画）、眼神如火 |
| **服饰** | 赵青灰深衣+**全甲护肩**、武冠、**披风** |
| **姿态** | 双手抱胸或**握长戟** |
| **背景** | 赵长城烽火台、军阵剪影 |

### 3.5 孟尝君（齐 · 谨慎渔利派）

| 维度 | 关键词 |
|---|---|
| **职能** | 齐的渔利谋主，门客众多 |
| **性格** | 老谋深算、世故、笑里藏刀 |
| **视觉** | 圆润、慈眉善目表象下眼神锐利、有留须 |
| **服饰** | 齐暖黄深衣+狐裘领、**手持算筹或玉珠** |
| **姿态** | 斜倚，单手拨算筹 |
| **背景** | 临淄市集门客剪影、账册堆 |
