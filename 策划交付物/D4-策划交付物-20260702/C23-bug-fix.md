## 二、Bug #1 修复 · 当前立场显示

### 2.1 问题

游戏过程中玩家无法看到自己"当前立场"——只能在终局时看到"你是【合纵派】"标题。

### 2.2 修复方案

**状态栏新增"立场指示器"**——每回合显示当前累计立场分（合纵 X / 中立 Y / 亲秦 Z）。

**UI 设计**（程序照搬）：

```
┌──────────────────────────────────┐
│ 合纵 4  │ 中立 1  │ 亲秦 1        │   ← 新增 · 立场指示器
├──────────────────────────────────┤
│ 玩家三维：合纵 50 / 名望 50 / 心计 40 │
│ 各国：秦 70/30/60 · 赵 50/60/40 · 齐 55/45/25 │
└──────────────────────────────────┘
```

**颜色编码**（与 C05 §三.2.4 立场按钮一致）：
- 合纵 = 青绿
- 中立 = 中灰
- 亲秦 = 黑红

**当前最高分的立场**加粗显示。

### 2.3 字段名

| 字段 | 来源 | 类型 |
|---|---|---|
| `stance_score` | C05 §2.1 Player | `Dictionary<String, int>`（key = hezong/zhongli/qinqin） |
| `stance_indicator_visible` | C23 新增 | bool（默认 true · 可在设置中关闭） |

### 2.4 修复状态

- ✅ 策划：UI 设计图就位
- ✅ 程序：UI 装配（10b ENG-06）
- ⏳ 美术：指示器颜色（10c ART-19）
