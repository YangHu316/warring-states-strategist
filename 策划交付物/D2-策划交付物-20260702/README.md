## 四、v7.3.5 适配要点（重要）

本目录所有 mock 决策表 + 召见相关产出均按 v7.3.5 面谈机制适配：

| v7.3.3 旧版 | v7.3.5 新版 | 影响文件 |
|---|---|---|
| 面谈 LLM 裁决 `{comprehension, stance_match, persuasion, verdict}` 综合分 → 采纳/拒绝 | 面谈君主 Agent 简述 + 提一个 `proposed_action` → 玩家以**三立场**（推合纵/中立/推亲秦）表态 | C12、C13 |
| mock 召见裁决表（综合分 6.5/4.0/5.0） | mock 立场判读（关键词 → 立场 → 触发 proposed_action 数值变动） | C13 |
| mock 决策表无 proposed_action 字段 | mock 决策表需支持"召见状态"分支 → 返回 `proposed_action` 字段 | C09–C11 |

详见各文件 §"v7.3.5 适配说明"小节。
