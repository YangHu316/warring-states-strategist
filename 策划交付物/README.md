## 五、文件命名规范

- **C 编号** = 策划文档编号（C01–C26，按 D1→D4 顺序）
- **RFC 编号** = 跨阶段修订记录（RFC-001, RFC-002）
- **推演报告** = `推演验证报告-v{X.X.X}-RFC{XXX}实施后.md`
- **质量报告** = `反向检测报告-{YYYYMMDD}.md` / `修复总报告-v{X.X.X}-{YYYYMMDD}.md`
- **集成指南** = `AI_INTEGRATION_GUIDE.md`

> **稳定字段名**（C05 §五 锁定）：所有 `Card.type` / `Card.direction` / `Card.target_country` / `Agent.action` / `Agent.settle_decision` / `Country.{guowei,mengxin,zhanxin}` / `Player.{hezong,mingwang,xinji}` 等**统一英文**，中文别名仅作注释。
